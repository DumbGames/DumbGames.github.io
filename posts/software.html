Software I use

I mainly use Free/Libre software, inshort software that respects user's freedom.

Operating system

I use a distribution of GNU/Linux, Artix Linux . Artix is based on Arch Linux which a very popular linux distro, So it is essentially arch but without systemD, an init system (first program to run as soon as you turn on your computer) that is kind of controversial, but I don't have any problem with systemD I just use it to look cool on the internet lol. And for the desktop environment I use the aweosme window manager, I don't use DWM or Xmonad like the cool kids, but I think I'll make a switch in the future.

Web browser

I use Libre-Wolf as my web browser. It is a fork of Firefox and provides state of the art privacy features out of the box. I have also installed some extentions like UBlock Origin for blocking trackers and ads, Decentraleyes for blocking connection to major CDN services for privacy, NoScript for blocking javascript and Dark Reader for my eyes.

Game laucher

I use lutris for launching my games and keeping them organized.

Text editor

I use both GNU Emacs and vim . I use Emacs for programming and other tasks where I have to keep staring at text for long and Vim for short text edits.
